# Referencias — Lead Dev AI

Esta carpeta contiene material de contexto para continuar madurando el descriptor de cargo.

## Qué poner aquí

- **JDs de referencia del mercado**: descriptores similares de otras empresas (LinkedIn, GetOnBrd, etc.)
- **Benchmarks salariales**: datos de mercado para validar el rango CLP $5M–$8M
- **Org chart**: estructura del Iron-Legion actual y planificada
- **Notas de hiring manager**: contexto sobre qué se busca realmente (SRM, fundadores)
- **Evaluaciones de candidatos previos**: qué funcionó y qué no en procesos anteriores
- **Role cards / RACI**: documentos operativos de los roles Technical Lead y SDM
